# Daily Planner

این پروژه یک برنامه‌ریز روزانه است که:
- امکان مدیریت برنامه روزانه برای هر روز هفته را دارد.
- با استفاده از API چت‌جی‌پی‌تی، برنامه پیشنهادی برای روز ارائه می‌دهد.
- امکان تغییر رنگ پس‌زمینه از بین ۳۰ رنگ مختلف را فراهم می‌کند.
- قابلیت تیک زدن کارهای انجام‌شده را دارد.
